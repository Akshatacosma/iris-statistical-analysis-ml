# Iris Dataset: Statistical Analysis and Machine Learning Classification

## Overview
This project demonstrates the application of statistical and probability concepts in machine learning using the Iris dataset. It includes data analysis, visualization, and supervised classification.

## Dataset
- Iris dataset from Scikit-learn
- Features: Sepal length, sepal width, petal length, petal width
- Target classes: Setosa, Versicolor, Virginica

## Statistical Analysis
- Descriptive statistics (mean, standard deviation)
- Feature distribution analysis
- Correlation analysis between features

## Machine Learning
- Model: Logistic Regression
- Feature scaling using StandardScaler
- Train-test split

## Model Evaluation
- Accuracy score
- Confusion matrix
- Precision, recall, and F1-score

## Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib & Seaborn
- Scikit-learn

## Conclusion
This project highlights how statistical analysis and probability concepts support effective machine learning model development and evaluation.
